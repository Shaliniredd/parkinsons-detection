🧠 Parkinson's Disease Detector

A machine learning web app that predicts Parkinson's disease from voice biomarker measurements.
Built with Python, Scikit-learn (SVM), and Streamlit.

📁 Project Structure

parkinsons_detection/
├── app.py                  ← Streamlit web app (main file)
├── 1_download_and_eda.py   ← Stage 1+2: Data download & EDA (run locally)
├── 2_train_model.py        ← Stage 3: Model training (run locally)
├── requirements.txt        ← Python dependencies for Streamlit Cloud
├── model/                  ← Saved model files (generated by training script)
│   ├── parkinsons_model.pkl
│   ├── scaler.pkl
│   └── feature_names.pkl
└── README.md
